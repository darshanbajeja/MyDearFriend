# Markdown Syntax Cheat-sheet

### Headings
```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```
### Text Formatting
```markdown
**Bold**
__Bold__
*Italic*
_Italic_
<u> Underline </u>
~~Strike~~
```
### Horizontal Rules
```markdown
---
___
```
---


### Links and Images

```markdown
[Title](Link)
![Alternative Text](link Title)
```
![Alternative Text]()

### Comments
```markdown
<!-- This is a comment -->
```

### Lists
```markdown
1. Item 1
2. Item 2

* Unordered List Item 1
* Unordered 2
```
